# 💸 Ploutos — Your AI Wealth Architect

> *"Financial therapy disguised as entertainment."*

Ploutos is a premium, AI-powered personal finance web app that feels like your **financially responsible best friend** — supportive, brutally honest, and funny enough that you *actually* want to open it daily.

## ✨ Features

### 🧠 Smart Financial Profile Builder
- Collects salary, expenses, debts, savings, investments, goals & risk comfort
- Generates a dynamic **Financial Health Score**
- Classifies you into a life stage: Teen Explorer, Early Earner, Wealth Builder, Stability Planner, or Retirement Strategist

### 🤖 AI Financial Coach Chatbot
- 7 personality modes: **Chaos Gremlin**, Rich Bestie, Gym Bro Investor, Soft Therapist, Corporate Villain, Delulu Motivator, Strict Indian Parent
- Answers questions like "What is SIP?", "Should I close my credit card debt first?", "Can I afford a car loan?"
- **Regional Language Support**: English, Hindi, Tamil, Telugu, Kannada, Bengali
- **Voice Input** simulation

### 📊 Visual Dashboards & Graphs
- Mental Stability vs Bank Balance chart (self-aware metrics)
- **Wealth Multiverse** — compare Investor You vs Chaotic You timelines
- Financial Aura indicator (Stable, Chaotic, Delusional, Wealthy)
- **Money Burn Rate** — "Your bank account survives X days"

### 🎯 Goal-Based Planning & Saving Jars
- Create custom savings jars for any goal
- Track contributions and progress visually
- Auto-integrates with Emergency Fund Planner

### 💀 Dark Humor Features
- **Monthly Financial Trauma Recap** (Spotify Wrapped-style)
- **Delusion vs Reality Meter** with Rich People Delusion Simulator
- **Biggest Villain Purchases** tracker
- Financial Situationship Status: *"Healing Era"*, *"Main Character Wealth Arc"*, etc.

### 🧮 Simulators & Tools
- **Debt Reduction Optimizer**: Snowball vs Avalanche comparison
- **AI Scenario Simulator**: "What happens if I change jobs / take a loan / start a business?"
- **Passive Income Planner** with 10-year cash flow projection
- **Emergency Fund Milestone** tracker

### 🏆 Gamification
- Daily streaks & XP system
- Achievement badges (New Egg, First Seed, Cafe Slayer, Week Fire...)
- Saving Challenges (No-Spend Week, Ditch Cafe Coffee, etc.)

### 👨‍👩‍👧 Couple / Family Finance Mode
- Shared budgeting toggle
- Joint savings jars with individual contribution tracking
- Partner Money Aura display

### 🌗 Theme Modes
- **Dark Mode** (glassmorphism neon, default)
- **Light Mode** (minimalist luxury)
- **AI Intervention / Calm Mode** (auto-triggers when finances are critical)

## 🚀 Getting Started

No build tools needed. Just open `index.html` in your browser or serve it with any static server:

```bash
# Option 1: Python
python -m http.server 8080

# Option 2: Node.js
npx serve .

# Option 3: Just open index.html directly in your browser
```
